# HabiNest📅✅

A simple and interactive **Habit Tracker Web Application** designed to help users build consistency, monitor daily habits, and improve productivity. The application provides an organized way to manage habits, track completion status, and view progress over time.

## 🚀 Project Overview

The **Habit Tracker** is a frontend-based productivity application developed using **HTML, CSS, and JavaScript**. It allows users to create daily habits, mark habits as completed, navigate through dates, and maintain a record of their progress.

The goal of this project is to encourage users to develop positive routines through an easy-to-use and visually clean interface.

---

## ✨ Features

* ✅ Add and manage daily habits
* ✅ Mark habits as completed


* ✅ Track daily progress
* ✅ Weekly calendar view
* ✅ Navigate between previous and next weeks
* ✅ Monthly date tracking
* ✅ Responsive design for desktop and mobile devices
* ✅ Interactive user interface

## ⚙️ How to Run the Project

### Method 1: Open Directly

1. Download or clone this repository.
2. Open the project folder.
3. Open:

```
index.html
```

in any browser.

---

### Method 2: Using VS Code Live Server

1. Install the **Live Server** extension in VS Code.
2. Open the project folder.
3. Right-click on `index.html`.
4. Select:

```
Open with Live Server
```

The application will open in your browser.
